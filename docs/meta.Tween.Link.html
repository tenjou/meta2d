meta.Tween.Link

new meta.Tween.Link()

Tweening library. Have dependency on Entity.Controller.
Properties:
Name Type Description
tween Object Tween who owns this link.
startValues Object Starting values of owner before tween.
endValues Object End values of owner after tween.
duration Number Duration of tween in milliseconds.
tDelay Number Wait time in milliseconds before next link is chosen.
tFrameDelay Number Wait time in milliseconds between frame update.
